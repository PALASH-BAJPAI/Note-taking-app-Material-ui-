## Installation 

Install project dependencies using

```bash 
  npm install
```

To run a json server

```bash
    npm install -g json-server
    json-server --watch data/db.json --port 8000
```


## Documentation

Features available:-

* Add a note
* Delete a note
* View all note
* Save the notes permanently
